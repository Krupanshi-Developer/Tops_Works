2. Difference Between git fetch and git pull

Answer:

git fetch downloads latest changes from GitHub but does not update local files.
git pull downloads changes and directly updates the local repository.

4. Difference Between Fork and Pull Request

Answer:

A Fork creates a copy of another repository in your GitHub account.
A Pull Request is used to request merging your changes into the original repository.
