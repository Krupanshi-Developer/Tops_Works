Session 1 :

1. Five IT Job Roles:

Answer:

Developer
 – Creates software, websites, and applications using programming languages.
Tester
 – Checks software for bugs and ensures everything works properly.
Designer
 – Designs the look and user experience of websites or apps.
Data Analyst
 – Studies data and creates reports to help companies make decisions.
AI Engineer 
– Builds smart systems using Artificial Intelligence and Machine Learning.

3. Full Forms and Uses

Answer:

GitHub
 – Platform used to store and manage code projects online.
Slack
 – Team communication and messaging application.
Jira
 – Project management and bug tracking tool.
IDE (Integrated Development Environment)
 – Software used to write and run code.
API (Application Programming Interface)
 – Helps two applications communicate with each other.

5. Difference Between Developer and Tester

Answer:

A developer creates and writes the software or application.
A tester checks the software for errors and makes sure it works correctly.
